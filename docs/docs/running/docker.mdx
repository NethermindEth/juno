# Building

```
docker build . -t juno
```

# Running

Create your config file and update it to the relevant values, see [Configuration](/docs/running/config) for more details.

You can then run your app using docker with:

```
docker run -p 6060:6060 -p 9090:9090 -v /local/path/for/db:/path/to/db/inside/container juno --network 1 --db-path /path/to/db/inside/container
```

For more flags, and usage, see the [Configuration](/docs/running/config) section.

## Open ports

By default, Juno uses ports 6060 and 9090 for the [RPC server](/docs/features/rpc) and the [metrics](/docs/features/metrics) respectively.

Remember to run the container with the `-p 6060:6060 -p 9090:9090 ` flags to enable these ports.
