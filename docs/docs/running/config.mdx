To quickly edit the configuration file, you can use the following command:

```
./build/juno configure
```

This will open a Terminal User Interface displaying the current configuration and allowing you to edit them.
Note that the command will overwrite your existing configuration with your changes.

# juno.yaml

Each of the configuration options has a corresponding CLI flag and environment variable.
For example, the CLI flag for the database path is `--database-path` and the environment variable is `JUNO_DATABASE_PATH` (all environment variables are prefixed by `JUNO_` to avoid collisions).

To see a description of the configuration options, run `./build/juno --help`.

Structure:
```yaml
log:
  level: INFO
  json: false
  nocolor: false
rpc:
  enable: false
  port: 8080
metrics:
  enable: false
  port: 9090
rest:
  enable: false
  port: 8100
  prefix: /feeder_gateway
database:
  path: /path/to/database
sync:
  enable: false
  sequencer: https://alpha-mainnet.starknet.io
  network: mainnet
  trusted: false
  ethnode: ""
```
