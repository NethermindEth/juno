Juno uses `yaml` format to for its configuration file. It can be provided via the following flag:
```bash
$ ./build/juno --config "path/to/config/file"
```

# juno.yaml

Each of the configuration options has a corresponding CLI flag. For example, the CLI flag for
the database path is `--db-path`.

To see a description of the configuration options, run `./build/juno --help`.

Example `juno.yaml`:
```yaml
verbosity: "debug"
rpc-port: 4576
metrics: true
db-path: "/home/.juno"
network: 1
eth-node: "https://some-ethnode:5673"
```
